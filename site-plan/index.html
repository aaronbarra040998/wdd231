Site Plan: Utah Trail Finder

WDD231 Individual Project Plan

1. Site Name

Utah Trail Finder

This name clearly communicates the site's purpose: helping users discover hiking trails in Utah. It's descriptive, memorable, and reinforces the site's geographic focus. The domain "utahtrailfinder.org" would be available for registration.

2. Site Purpose

The site provides a comprehensive hub for Utah hiking enthusiasts by offering:

  • Searchable database of 50+ hiking trails with detailed information
  • Real-time weather integration for trail conditions
  • Interactive trail difficulty ratings and user reviews
  • Trail report submission form for community contributions
  • Responsive trail maps and elevation profiles
  • Personal trail wishlist saved via local storage

3. Scenarios

  • Scenario 1: "I'm a beginner hiker with young kids. What are the easiest, shortest trails near Provo that have good views?"
  • Scenario 2: "I'm planning to hike Mount Timpanogos this weekend. What's the current weather forecast and trail condition?"
  • Scenario 3: "I've hiked the Bright Angel Trail and want to submit a report about muddy conditions near the saddle. How do I share this information?"

4. Color Schema

Body text, captions, secondary info

Sky Blue

#4A90E2

Links, accents, hover states, decorative elements

Cream White

#F5F5DC

Background, content containers

5. Typography

Montserrat (Bold 700)

Used for all page titles, section headings, and navigation.

Provides strong visual hierarchy and modern appeal. Excellent readability at all sizes.

Open Sans (Regular 400, SemiBold 600)

Used for all body text, paragraphs, list items, and form elements.

Ensures excellent readability across devices with its clean, neutral design.

6. Wireframes

Footer Links
